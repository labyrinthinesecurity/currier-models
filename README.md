This repository contains details support material for https://arxiv.org/pdf/2604.25979 preprint

All scripts use RF1b-e.txt for VMS transliteration.

# Part 1 (Machine learning generation models of the A/B split)

- generative models: unsupervised.py and its report: unsupervised_report.txt
- binomial models (generative plus predictive): generative_predictive.py and its report: generative_predictive_report.txt

# Part 2 (The A/B contrast is not primitive)

- e/ch anomaly: e_ch_anomaly.py and its report: e_ch_anomaly_report.txt
- evidence of the 3 layers structure sustaining the A/B contrast: switch.py and its report: switch_report.txt
